# Insumia — Guia de Acesso para Teste

Bem-vindo! Este guia mostra como acessar o app e o back-office da Insumia para testar
o fluxo completo: uma clínica faz um pedido no app → você vê o pedido chegar no
back-office como administrador.

## 1. Instalar o app no iPhone (TestFlight)

1. Instale o app **TestFlight** pela App Store (gratuito, ícone azul).
2. Você vai receber um **e-mail da Apple** chamado *"You're invited to test Insumia"*.
3. Abra o e-mail no iPhone e toque em **"View in TestFlight"** (botão azul).
4. O TestFlight abre direto — toque em **Instalar** ao lado de **Insumia**.
5. Quando terminar, abra o **Insumia** pelo ícone (não pelo TestFlight).

> Pra atualizações futuras do app é só abrir o TestFlight; quando houver versão nova, aparece o botão **Atualizar**.

---

## 2. Instalar o app no Android (APK direto)

1. Toque neste link no celular Android: **(cole aqui o link do .apk)**
2. O navegador baixa o arquivo `Insumia.apk`.
3. Toque no arquivo baixado.
4. Quando aparecer **"Permitir instalações desta fonte"**, toque em **Permitir**.
5. Toque em **Instalar** → **Abrir**.

> Insumia vira app no menu, abre direto.

### Como fazer um pedido

1. Faça login com uma das clínicas acima.
2. Na tela inicial, toque em **Solicitar** (ou no botão **+** no rodapé).
3. Busque os medicamentos e toque para adicioná-los ao pedido.
4. Toque em **Ver Pedido**.
5. Ajuste as quantidades e toque em **Enviar Pedido**.

A clínica acabou de mandar o pedido para a Insumia.

> Para trocar de clínica: aba **Menu** → **Sair** → entrar com outro e-mail.

## 4. Ver o pedido chegar no back-office (lado do admin)

O back-office é o painel onde a equipe Insumia gerencia todos os pedidos.

1. Acesse: **https://insumiaapp.bahflash.tech**
2. Faça login com a conta admin:
   - **E-mail:** `valerio@insumia.app`
   - **Senha:** `demo12345`
3. Na lateral, vá em **Pedidos**.
4. O pedido que você acabou de fazer no app aparece no topo — com o nome da clínica
   que o enviou.

A partir daí, o admin pode analisar, agrupar pedidos por medicamento e montar a
cotação — o ciclo de compra coletiva da Insumia.

---

## Fluxo do teste, em um minuto

1. **No celular**: entra numa clínica → faz um pedido.
2. **No navegador**: entra como Valério → aba Pedidos.
3. O pedido aparece lá, vindo da clínica que enviou.

Pronto, ciclo completo.
